🔥 Firebase 연결 설정

아래 3단계를 따라하면 팀원 전원이 같은 링크에서 실시간 편집할 수 있어요.
최초 1회만 설정하면 이후엔 자동으로 연결됩니다.

① Firebase 프로젝트 만들기
console.firebase.google.com → 프로젝트 추가 → 이름 입력 → 계속

② Realtime Database 생성
좌측 메뉴 → Realtime Database → 데이터베이스 만들기 → 테스트 모드 선택 → 완료

③ 규칙(Rules) 설정
Rules 탭 클릭 → 아래 내용으로 교체 후 게시:
{ "rules": { ".read": true, ".write": true } }

④ Database URL 복사
데이터 탭 상단 URL (예: https://my-app-default-rtdb.firebaseio.com)

입력한 URL은 이 브라우저에만 저장됩니다. 팀원들도 같은 URL을 입력하면 동일한 데이터를 공유할 수 있어요.
